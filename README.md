TraceLens AI

AI-Powered Crime Investigation and Digital Evidence Analysis

Overview

TraceLens AI is an intelligent crime investigation platform designed to assist law enforcement agencies in analyzing digital evidence. The application integrates AI-based anomaly detection, communication timeline analysis, network visualization, and automated investigation reports to help investigators identify suspicious activities efficiently.

Features

- AI-powered anomaly detection
- Call and chat record analysis
- Communication timeline visualization
- Network graph of contacts
- AI-generated investigation insights
- Interactive dashboard using Streamlit
- Automated case summary generation

Tech Stack

- Python
- Streamlit
- Pandas
- Matplotlib
- NetworkX
- Google Gemini API

Installation

1. Clone the repository

git clone https://github.com/your-username/TraceLensAI.git

2. Navigate to the project directory

cd TraceLensAI

3. Install dependencies

pip install -r requirements.txt

4. Run the application

streamlit run app.py
